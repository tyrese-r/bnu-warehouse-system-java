# Warehouse Management System

Java warehouse management system
## Requirements

- Java 24 or higher
- Gradle 8.14 or higher

## Running the Application


```bash
./gradlew run
```

## Testing

```bash
./gradlew test
```

## Compiling

To compile the application without running it:

```bash
./gradlew compileJava
```

```bash
./gradlew build
```
